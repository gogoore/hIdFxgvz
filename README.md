# 项目简介

## 核心功能
- 基于YOLOv8的垃圾桶溢满检测系统
- 支持5000张数据集，包含'garbage'、'garbage_bin'、'overflow'三类图像
- 支持图片、视频及摄像头进行实时检测
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择不同的检测模型

## 适用场景
- 智能环卫管理系统
- 垃圾分类与收集监控
- 城市环境智能监测
- 公共卫生管理

## 技术特点
- 基于Python 3.9开发
- 使用PyQt5构建用户界面
- 集成OpenCV和PyTorch框架
- 预训练模型可直接使用
- 提供完整源码和运行环境配置
